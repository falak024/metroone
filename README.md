METROONE- (DELHI METRO ROUTE FINDER)
A Java based Delhi Metro Route Finder using Dijikstra's algorithm and Swing GUI.

FEATURES:
Shortest path using dijikstra algo
Fare estimation
Travel time calculation
30 major delhi metro stations
Interchange detection
Interactive Swing GUI

*TECHNOLOGIES USED*
Java
Java Swings
Dijikstra algorithm
Data Structures( Graphs, Priority Queue)
OOP

HOW TO RUN-
1. Clone the repository
2. Open the project in IntelliJ/Eclipse/VS Code
3. Run Main.java
